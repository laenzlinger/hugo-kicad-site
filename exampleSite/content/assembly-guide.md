## Before You Start

Make sure you have all components from the BOM. A hot-air station is recommended
for the QFN packages.

## Step 1: Power Section

Start with the voltage regulators (U1, U2) and their surrounding passives.
Test the 3.3V and 5V rails before proceeding.

## Step 2: Compute Module

Solder the CM4 connectors. Do not install the CM4 module yet.

## Step 3: Audio Path

Install the relay drivers and audio jacks last to avoid mechanical stress
during soldering.
