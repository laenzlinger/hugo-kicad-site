## About this example

This page demonstrates the **board** layout type, which combines:

1. An interactive KiCanvas schematic/PCB viewer (above)
2. A 3D render gallery (when renders are available)
3. An interactive BOM viewer (when iBOM is available)
4. Custom markdown content (this section)

The KiCad files shown here are from the **Arduino Uno Shield** template included with KiCad.

In a real project, 3D renders and the interactive BOM are generated automatically by [KiBot](https://github.com/INTI-CMNB/KiBot) during CI.
